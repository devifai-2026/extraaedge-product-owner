# extraaedge-product-owner

Product Owner control plane for the extraaedge platform. Manages tenants,
platform users, plans, audit log, and the cross-tenant Support Tickets inbox.

## Run

```bash
npm install
npm run dev      # http://localhost:5174
```

Set `VITE_API_BASE_URL` in `.env` (defaults to `http://localhost:4000/api/v1`).
